# Get parameters from SSM parameter store via NodeJS

This script retrieves the following parameter types:
- String
- StringList
- SecureString

## Tested on
[NodeJS v20.2.0](https://nodejs.org/en/blog/release/v20.2.0)

## Run
To run the script:
```bash
npm install
node index.js
```
